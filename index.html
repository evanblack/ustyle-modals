uSwitch modal dialog script.

Yet another modal dialog script, but made for uSwitch with the sort of options we need, and especially with responsiveness in mind.

This hosted page is purely for examples. For requirements, installation and usage, see the GitHub repo.

Please note if you're using IE9+ or Firefox, you will not see custom web fonts (it's a CORS issue). Also that debugging is on (the HTML tag has a class of "us-debug"), so the JS console will be delivering lots of messages.

JS triggered

You can also trigger a modal programmatically in JS, with exactly the same options.

  • JS triggered (all the options!)
$('#js-example1').click(function(){
  uSwitch.modal.openModal(false,{
    'type':'ajax',
    'target':'ajax.html',
    'width':'xl',
    'title':'JS triggered',
    'modalclass':'green-footer',
    'closedirection':'right',
    'formhtml':'<h3>Don\'t delay...</h3>',
    'formurl':'#',
    'forminputs':{
        'HIDDEN-INPUT-1':{
          'type':'hidden',
          'value':'hidden!!'
        },
        'HIDDEN-INPUT-2':{
          'type':'hidden',
          'value':'also hidden!!'
        }
      },
    'formbutton': {
      'class':'btn_green btn_arrowed',
      'text':'click now!'
    },
    'formhtml':'<h3>Don\'t delay...</h3>'
  });
})
