uSwitch modal dialog script.

Yet another modal dialog script, but made for uSwitch with the sort of options we need, and especially with responsiveness in mind.

This hosted page is purely for examples. For requirements, installation and usage, see the GitHub repo.

Please note if you're using IE9+ or Firefox, you will not see custom web fonts (it's a CORS issue).

Examples

Link triggered Non-JS

Not using any JS, only CSS. Requires the modals to be prebuilt and hidden in the page.

Link triggered JS delegated event

In this mode, you can just add a class to the link you want to trigger the modal.
